# Proyecto Kevin — Repositorio de aprendizaje Full Stack

Repositorio maestro para documentar y demostrar el aprendizaje del stack
del Proyecto Kevin.

## Objetivo

Desarrollar progresivamente una plataforma web profesional utilizando
fundamentos web, TypeScript, React, Next.js, PostgreSQL, Prisma, analítica,
SEO, seguridad, testing y despliegue.

## Requisitos

- Node.js 24 LTS
- npm
- Git

## Verificar el entorno

```bash
npm run doctor
```
